parse(binary)

Specs:

  • parse(binary) :: {float, binary} | :error

Parses a binary into a float.

If successful, returns a tuple of the form { float, remainder_of_binary }. Otherwise :error.

Examples

iex> Float.parse("34")
{34.0,""}
iex> Float.parse("34.25")
{34.25,""}
iex> Float.parse("56.5xyz")
{56.5,"xyz"}
iex> Float.parse("pi")
:error
