Ecto.Migrator

This module provides the migration API.

Example

defmodule MyApp.MigrationExample do
  use Ecto.Migration

  def up do
    "CREATE TABLE user(id serial PRIMARY_KEY, username varchar(25));"
  end

  def down do
    "DROP TABLE user;"
  end
end
